# Routes

| Route | Method | Description |
| :--- | :--- | :--- |
| /token | POST | Delete post. |
| / | POST | Add one user. |
| /users/me | GET | Get current user. |
| /posts | GET | List posts. |
| /posts/filter | POST | Filter posts. |
| /posts | POST | Add one post. |
| /posts/:id | DELETE | Delete post. |
